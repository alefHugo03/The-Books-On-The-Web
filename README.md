Objetivo:

    Criação de um site de contratação de livros voltados para a área de TI.

Como foi usado:

    Foi usado JS para:
    
    Enviar a requisição ao servidor
    Dinamiticidade das paginas como a criação da capa do livro

    Fois usado o PHP para

    Fazer requisições ao banco de dado
    Fazer o CRUD dos dados e dos livros
    Proteção da pagina
