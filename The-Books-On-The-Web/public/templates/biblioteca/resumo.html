Projeto Universitário

Desenvolvido no segundo semestre de 2025, este projeto visa democratizar o acesso a materiais de estudo na área de tecnologia. Nosso objetivo é facilitar a pesquisa e o acesso a livros técnicos que, no mercado físico, possuem valor elevado.

Metodologia de Coleta

O acervo é curado através de repositórios open-source no GitHub que disponibilizam conteúdo educacional. Nossa plataforma organiza esses materiais brutos em uma interface amigável, com capas, sinopses e leitura facilitada, transformando repositórios de código em uma biblioteca digital visual.
